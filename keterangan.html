Keterangan Sistem

Tentang Sistem

Sistem informasi pengelolaan satuan pendidikan non formal SKB/PKBM tahun 2026 di lingkungan Dinas Pendidikan Kabupaten Sumedang. Versi statis ini di-host di GitHub Pages dan menyimpan data secara lokal di browser pengguna (localStorage).

Kategori Data

  1. Data Akreditasi – Daftar status akreditasi seluruh lembaga PKBM/SKB beserta tahun berlaku dan nomor SK.
  2. Data Peserta Didik SPMB ATS/APS – Rekapitulasi peserta didik baru jalur ATS/APS, termasuk komposisi gender dan status verval.
  3. Data E-Rapor 8 SNP SKB/PKBM – Capaian 8 Standar Nasional Pendidikan (SKL, Isi, Proses, Penilaian, PTK, Sapras, Pengelolaan, Pembiayaan).

Fitur

  • Grafik laporan otomatis di Beranda untuk 3 kategori data.
  • Tombol + Input Data dengan opsi pilihan (dropdown) yang sesuai konteks.
  • Tombol Edit & Hapus per baris.
  • Tombol Cetak (print-friendly) & Unduh PDF (jsPDF).

Legenda Warna

● Baik / Sudah / Berlaku
● Cukup / Sedang
● Kurang / Belum / Habis Berlaku
● Informasi / Tautan

Reset Data

Klik tombol di bawah untuk menghapus semua data lokal dan mengembalikan ke data contoh awal.
